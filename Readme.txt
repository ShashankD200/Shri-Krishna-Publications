# SHRI KRISHNA PUBLICATIONS- RAIPUR

This repository contains the source code for the SHRI KRISHNA PUBLICATIONS- RAIPUR website. The website is designed to be static with a dynamic "Contact Us" section. It is built using HTML, MySQL, AJAX, jQuery, and Bootstrap.

## Project Overview

This website serves as a promotional platform for SHRI KRISHNA PUBLICATIONS- RAIPUR. It provides information about the publications and includes a dynamic contact form for users to get in touch.

## Technologies Used

- HTML
- MySQL
- AJAX
- jQuery
- Bootstrap

## Features

- Static pages for promotional content.
- Dynamic "Contact Us" section for user interaction.
- Utilizes AJAX for seamless form submissions.
- Responsive design using Bootstrap for optimal viewing on various devices.
